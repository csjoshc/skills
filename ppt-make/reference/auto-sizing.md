## Body Font by Content Density

| Content level                | Body font | Shape labels |
|------------------------------|-----------|--------------|
| Light (<=4 lines, <200 chars)| 16pt      | 12pt bold    |
| Medium (5-7 lines)           | 14pt      | 11pt bold    |
| Dense (8-10 lines)           | 12pt      | 10pt bold    |
| Very dense (>10 lines)       | 11pt      | 9pt bold     |

## Box Sizing by Count

| Box count | Width       | Height      | Gap          |
|-----------|-------------|-------------|--------------|
| <=5       | 2.00-2.50"  | 0.85-0.90"  | 0.20-0.40"   |
| 6-8       | 1.50-2.00"  | 0.70-0.85"  | 0.15-0.25"   |
| >8        | 1.00-1.50"  | 0.58-0.70"  | 0.10-0.15"   |

## Table Font by Density

| Columns x Rows      | Body font | Header font |
|----------------------|-----------|-------------|
| <4 cols, <6 rows    | 11pt      | 10pt        |
| 4+ cols or 6+ rows  | 10pt      | 9pt         |
| 5+ cols or 8+ rows  | 9pt       | 8pt         |

## Space Utilization

- Slide dimensions: 13.33" x 7.50" (16:9 widescreen)
- Footer zone: bottom 0.35" reserved (copyright, logo)
- Content area: 0.30" to 12.75" horizontal, 0.18" to 7.10" vertical
- **Target 75-85% fill** — dark background amplifies empty space
- In-slide links: Y = 6.65-6.88" (above footer)
